# 🚀 Proyecto Portafolio Web - Jorge Leiva

## 📌 Descripción

Este proyecto corresponde a la modificación de una plantilla HTML basada en Bootstrap, donde se personalizó un portafolio web agregando información personal, habilidades y proyectos.

El objetivo fue aplicar conceptos básicos de Git y GitHub como:

* Fork
* Clonación
* Commits
* Push a repositorio remoto

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome

---
